# KotlinComposeSample                                                                                         
                                                                                                                                                                                                           
  An Android app built with Jetpack Compose demonstrating a health tracking UI.                                                                                                                            
   
  ## Features                                                                                                                                                                                              
  - Personal details screen
  - Blood pressure tracking screen
  - ViewModel-based state management                                                                                                                                                                       
   
  ## Tech Stack                                                                                                                                                                                            
  - Kotlin        
  - Jetpack Compose
  - MVVM (ViewModel)
  - Material 3                                                                                                                                                                                             
   
  ## Getting Started                                                                                                                                                                                       
                  
  ### Prerequisites
  - Android Studio Hedgehog or later
  - Android SDK 21+
                                                                                                                                                                                                           
  ### Run
  1. Clone the repo                                                                                                                                                                                        
     git clone https://github.com/RahZams/KotlinComposeSample.git
  2. Open in Android Studio                                                                                                                                                                                
  3. Run on emulator or device
